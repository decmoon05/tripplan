# TripWise — 프로젝트 계획서

> 이 문서는 프로젝트의 설계도입니다. 무엇을 만들지 처음부터 끝까지 기록합니다.
> 새 기능 작업 전 반드시 이 문서를 확인해 현재 Phase에 포함되는지 검증할 것.

---

## 제품 비전

**"내가 어떤 사람인지 알고, 나에게 딱 맞는 여행을 설계해주는 AI 여행 앱"**

기존 여행 앱의 문제를 해결:
- 획일적 추천 (다들 가는 패키지 코스) → **성격/취향 기반 개인화**
- 정보 부족한 UI → **정보량 제어 (간결 기본, 상세 선택)**
- 즉흥 변경 불가 → **실시간 대안 추천**
- 여러 앱 왔다갔다 → **올인원 (지도, 일정, 재정 통합)**

## Phase 1 — MVP (현재 목표)

**목표**: 핵심 가치 검증. 성격 기반 AI 추천이 정말 다른가?

### 기능 목록

#### 1. 사용자 온보딩 & 성격 프로파일링
- 고정 질문 12개 (MBTI 스타일, 여행 페이스, 음식 취향 등)
- AI가 답변 분석 후 보충 질문 2~4개 추가 (유기적)
- 질문 형식: 버튼 탭 (예/아니오/상관없음/때에 따라/직접 입력)
- 완성된 프로파일은 DB 저장 (재방문 시 재사용)

#### 2. 목적지 선택
- 검색 또는 인기 목적지 탭 선택
- 여행 날짜 선택 (미정 가능)
- 동행 유형 선택 (혼자/커플/친구/가족)

#### 3. AI 일정 초안 생성
- Claude API가 프로파일 + 목적지 기반으로 일정 생성
- 동선 최적화 (숙소 위치 기준, 이동 시간 최소화)
- 각 여행지마다: 이름, 카테고리, 추천 이유 (프로파일 연결), 예상 소요시간

#### 4. 여행지 정보 카드
- 기본 뷰: 이름, 카테고리 아이콘, 평점, 거리
- 상세 뷰 (탭으로 펼치기): 설명, 영업시간, 가격대, 주소
- Google Places API 데이터 기반

#### 5. 여행 전 실시간 이슈 체크
- 여행 일정 확정 시 자동 실행
- 확인 항목: 공사/폐업, 공휴일 휴무, 특수 이벤트, 안전 경보
- Claude API + 웹 검색 활용

## Phase 2 — 차별화 강화 (MVP 검증 후)

- 재정 바구니: 환율, 예상 비용 계산기, 가격 장바구니
- 동행 심화 프로파일: 찐친/가벼운친구/연애초반/중기 등
- 즉흥 변경 모드: 현위치 기반 근처 대안 실시간 추천
- 취미/마니아 매칭: 사격, 애니메이션, 특수 액티비티 연동
- 나이대별 소셜 추천: 20대/30대/4050대 인기 코스

---

## Phase 3 — 수익화 (사용자 확보 후)

- 숙소/항공 제휴 링크 (커미션 수익)
- 프리미엄 구독 (고급 개인화, 실시간 가격 알림)
- 사용자 기여 데이터 커뮤니티
- 기업/여행사 API 제공

## 핵심 화면 목록 (Phase 1)

```
1. 온보딩/프로파일링 화면
   ├── 질문 카드 슬라이드
   └── 프로파일 완성 화면

2. 홈 화면
   ├── 프로파일 요약
   ├── 새 여행 시작 버튼
   └── 이전 여행 기록

3. 목적지 선택 화면
   ├── 검색바
   ├── 인기 목적지 그리드
   └── 날짜/동행 선택

4. AI 일정 생성 로딩 화면
   └── 생성 중 애니메이션

5. 일정 뷰 화면 (핵심)
   ├── 날짜별 탭
   ├── 여행지 카드 리스트 (드래그로 순서 변경)
   ├── 지도 뷰 토글
   └── 여행지 상세 모달

6. 이슈 체크 화면
   └── 발견된 이슈 목록 + 대안 제안
```

```
trip_places: id, trip_day_id, place_id, order, duration_minutes, notes
places: id, google_place_id, name, category, lat, lng, cached_data(JSON), cache_expires_at
travel_issues: id, trip_id, place_id, issue_type, description, severity
```
