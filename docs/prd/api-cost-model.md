## 4. 비용 최적화 전략

### 즉시 적용
1. **Google API 필드 마스크:** 필요한 필드만 요청 (비용 절감)
2. **클라이언트 세션 캐시:** 동일 장소 30분 내 재조회 방지
3. **AI 응답 캐싱:** 동일 조건 일정은 캐시에서 제공 (DB에 결과 저장 가능)
4. **무료 사용 제한:** 월 1~2회 무료, 이후 유료

### 중기 적용
5. **하이브리드 AI 모델:** 단순 작업은 mini 모델, 복잡한 생성은 풀 모델
6. **배치 처리:** 비실시간 요청은 배치로 묶어 할인 적용
7. **Google Maps Platform 볼륨 할인:** 월 $10K+ 시 자동 적용
