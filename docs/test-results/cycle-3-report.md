## 새로 발견된 문제

### NEW-P1: 음수 시간 버그 (-10:-46)
- 24시간 경유, 심야 일정에서 시간 계산이 음수로 감
- `optimizeRouteOrder()`의 `cappedStart = Math.min(currentTime, 23*60+30)` — currentTime이 이미 음수면 캡이 안 걸림
- **수정**: `Math.max(0, currentTime)` 하한 추가 필요

### NEW-P2: 심야 시간대 미지원
- 23시~03시 영업 식당이 시간 시스템(00:00~23:59)과 충돌
- 라멘집이 25:00(=새벽 1시) 영업인데 시스템이 처리 못함

- 프롬프트 강화만으로는 해결 불가. **생성 후 저녁 보강 로직** 필요
