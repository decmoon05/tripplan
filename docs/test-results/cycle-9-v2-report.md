## 근본 원인 발견

### 🔴 핵심 버그: `skipDinner = day === maxDay`

`augmentMissingMeals()`에서 **마지막 날은 무조건 저녁 삽입을 스킵**하고 있었다.

```typescript
// 버그 코드 (line 192)
const skipDinner = day === maxDay;  // 3일 여행이면 Day 3 저녁 무조건 스킵
```

**영향**: 4/7 FAIL이 이 한 줄 때문이었다. 3일 여행(후쿠오카, 교토, 오키나와)은 모두 Day 3 저녁 누락.

### 수정

```typescript
// 수정 코드
const skipDinner = false;  // 출발 시간 정보 없으므로 항상 삽입 시도
```

마지막 날 오전 출발이면 AI가 저녁을 안 넣었을 것이고, augment도 일정이 15:00에 끝나면 17:00~21:30에 빈 슬롯이 있어서 자연스럽게 삽입된다.
