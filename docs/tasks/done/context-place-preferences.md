# 장소 경험 카드 + 환율 표시 맥락 노트

## 요구사항 원문
- 여행 계획 생성 전, 목적지 인기 장소를 카드로 보여주고 경험 선택
  - "가봐서 안갈래요" / "가봤는데 또갈래요" / "안가봤어요"
- 선택 데이터를 사용자 정보에 저장 → 다음 계획에도 재사용
- 모든 나라 화폐를 현지 통화 + 한화 환산으로 표시

## 설계 결정
- 환율: 우선 정적 테이블 → 추후 API 연동 (MVP 속도 우선)
- 인기 장소 리스트: AI에게 간단 호출로 생성 (별도 DB 불필요)
- place_name 기반 매칭 (place_id는 Google ToS 제약)
