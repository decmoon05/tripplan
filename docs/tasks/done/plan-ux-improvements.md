# UX 불편사항 7가지 개선 계획

## 목표
사용자가 실제 사용 중 발견한 UX 불편사항 7가지를 우선순위별로 해결

## 구현 순서

### 1차 (이번 세션)
1. **이슈 #5**: 대시보드 — 여행 목록 페이지 (`/dashboard`)
2. **이슈 #1**: 마이페이지 — 프로필 확인/수정 (`/mypage`)
3. **이슈 #3**: 자동 진행 — 옵션 클릭 시 다음 스텝 이동

### 2차
4. **이슈 #7**: 지도+타임라인 동시 표시 (데스크톱 분할)
5. **이슈 #6**: 비행기/숙소 고려 (프롬프트 + 입력필드)

### 3차
6. **이슈 #2**: 커스텀 관심사/식성 (DB 마이그레이션 포함)
7. **이슈 #4**: 장소 이미지/설명 (Google Photos API)

## 기술 결정
- 대시보드: `useTrips` 훅 + TripCard 컴포넌트
- 마이페이지: ProfileForm isEdit prop 추가로 재사용
- 자동 진행: handlePrefNext를 onChange에서 직접 호출
- 지도+타임라인: 데스크톱 grid-cols-2, 모바일 토글 유지
- 비행기/숙소: tripPreferencesSchema 확장 + 프롬프트 반영
