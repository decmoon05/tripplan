# 셀프 체크 리마인더

> 작업 완료 보고 전 아래 항목을 반드시 확인할 것.
> 모든 항목에 "예"로 답할 수 있어야 작업 완료로 인정한다.

## 필수 확인 항목

### 기능 완성도
- [ ] 요구된 기능이 모두 구현되었는가?
- [ ] 엣지 케이스를 고려했는가? (빈 값, null, 경계값)

### 에러 처리
- [ ] 실패할 수 있는 모든 작업에 에러 처리를 추가했는가?
- [ ] 에러 메시지가 사용자 친화적인가?
- [ ] 빈 catch 블록은 없는가?

### 보안
- [ ] 사용자 입력을 검증하고 있는가?
- [ ] 하드코딩된 비밀키/비밀번호는 없는가?
- [ ] SQL 인젝션, XSS 등의 취약점은 없는가?

### 코드 품질
- [ ] 네이밍 컨벤션을 준수했는가?
- [ ] 불필요한 console.log를 제거했는가?
- [ ] 타입이 정확하게 지정되었는가? (any 사용 없음)

### 작업 문서
- [ ] 체크리스트를 업데이트했는가?
- [ ] 중요한 결정 사항을 맥락 노트에 기록했는가?
